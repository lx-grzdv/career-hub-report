## 2. Файлы скриншотов

Ожидаемые файлы в `public/screenshots/`: `slice-YYYY-MM-DD-HH-MM.png` по каждому срезу.

- **Сейчас:** 11 срезов в `SNAPSHOTS`, 11 файлов в `public/screenshots/` — все на месте. Срезы без скриншотов из таблицы удалены.

## 3. Совпадение данных на скриншотах с таблицей

Чтобы проверить, что числа на каждом скриншоте совпадают с `members` в `snapshot.ts`:

1. Запустите dev-сервер: `npm run dev`
2. В другом терминале: `npx tsx scripts/verify-screenshots.ts --parse`
3. В `.env` должен быть задан `OPENAI_API_KEY` (разбор скриншотов идёт через OpenAI Vision).

Скрипт отправит каждый PNG из `public/screenshots/` в `/api/parse-snapshot-image`, получит распознанные числа и сравнит их с данными в `SNAPSHOTS`.

## Быстрая проверка (без API)

Только порядок и наличие файлов:

```bash
node scripts/verify-screenshots.mjs
```
